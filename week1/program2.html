Hyperlink Demonstration

The <a> tag is used to create hyperlinks in HTML.

Examples of Hyperlinks

  1. Visit Google (same tab)

    Explanation: The target="_self" attribute is the default and opens the link in the same tab.

  2. Visit Wikipedia (new tab)

    Explanation: The target="_blank" attribute opens the link in a new tab or window.

  3. Jump to Section 2 (same page)

    Explanation: Using # followed by an ID allows navigation to a specific section of the same page.
